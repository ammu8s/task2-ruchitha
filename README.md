# E-Commerce Exploratory Data Analysis System

## Overview

This project was developed as part of a Data Analytics training program to perform comprehensive Exploratory Data Analysis (EDA) on an E-Commerce dataset. The objective of the project is to analyze customer transactions, identify purchasing trends, understand sales patterns, and generate actionable business insights through data visualization and statistical analysis.

The project demonstrates the practical application of data analytics techniques, enabling better decision-making through data-driven insights.

## Features

* Data inspection and quality assessment
* Missing value analysis
* Duplicate record identification
* Statistical summary generation
* Univariate and multivariate analysis
* Outlier detection using visualizations
* Correlation analysis among numerical features
* Business insight generation through graphical representations

## Technologies Used

* Python
* Pandas

* Jupyter Notebook

## Analysis Performed

### Data Inspection

Performed an initial review of the dataset structure, data types, missing values, and duplicate records.

### Statistical Analysis

Generated descriptive statistics to understand the distribution and characteristics of numerical variables.

### Missing Value Analysis

Identified columns containing missing values and evaluated their impact on the dataset.

### Outlier Detection

Used box plots and statistical methods to identify unusual observations within numerical features.

### Correlation Analysis

Analyzed relationships among numerical variables using correlation matrices and heatmaps.

### Data Visualization

Created visual representations to better understand:

* Product sales distribution
* Customer purchasing behavior
* Payment method preferences
* Order status trends
* Revenue patterns
* Feature relationships
